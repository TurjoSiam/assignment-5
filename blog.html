Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface which works as a medium for linking html files with javascript so that the javascript can locate the different parts of the html file as objects and nodes to modify them and make the webpage dynamic and user friendly.

How do you select an element from the DOM?

We can select an element from the DOM 5 ways,

  1. getElementByTagName
  2. getElementById
  3. getElementsByClassName
  4. querySelectorAll
  5. querySelector

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a process of attaching an event listener to a parent element so that it can handle events for all of its children.


Usefulness: Instead of attaching a click handler to each item, event delegation allows to attach a single listener to the parent container. This results in less code and better performance, especially for large sets of elements. It is particularly useful when dealing with dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles at first we have to access the element using methods like getElementById, getElementsByClassName, or querySelector. For manipulating attributes we can use,

  • setAttribute()
  • getAttribute()
  • removeAttribute()

for manipulating styles, we can use,

  • classList.add()
  • classList.remove()
  • classList.replace()
  • classList.toggle()
  • classList.contains()
